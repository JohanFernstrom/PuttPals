## Getting Started

### Prerequisites

- Node.js (for development)
- Modern web browser

### Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/PuttPals.git
   cd PuttPals
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Start the development server:
   ```
   npm start
   ```

4. Open your browser to http://localhost:3000
